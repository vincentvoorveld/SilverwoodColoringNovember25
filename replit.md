# Silverwood Coloring - Static Website

## Overview
A complete static website for Silverwood Coloring, a brand promoting coloring books by Aria Silverwood sold on Amazon. Built with pure HTML, CSS, and vanilla JavaScript.

## Project Structure
```
/
├── index.html                 # Home page with hero section
├── series.html                # Series overview page
├── about.html                 # About Aria Silverwood
├── faq.html                   # Frequently asked questions
├── contact.html               # Contact and social links
├── sinterklaas.html          # Sinterklaas category page
├── stained-glass.html        # Stained Glass category page
├── bold-easy.html            # Bold and Easy category page
├── cities.html               # Cities category page
├── mandala-animals.html      # Mandala Animals category page
├── other.html                # Other Books category page
├── assets/
│   ├── css/
│   │   └── style.css         # Main stylesheet
│   └── images/               # Image assets (placeholders)
└── data/
    └── books.json            # Book catalog data
```

## Design Theme
- **Color Scheme**: Soft pastel colors (gentle pinks, light blues, warm yellows)
- **Background**: Off-white (#faf9f7)
- **Font**: Poppins (Google Fonts)
- **Style**: Playful and creative with rounded cards, subtle shadows, smooth hover effects
- **Responsive**: Fully responsive for desktop, tablet, and mobile

## Features
- Sticky navigation header with active page highlighting
- Dynamic book loading from JSON data
- SEO optimization with meta tags and Open Graph
- Lazy-loading images
- Ready for GitHub Pages deployment
- All relative paths for portability
